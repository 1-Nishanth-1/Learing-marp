# Math

<!-- _class: invert -->

- Worst case search in a list:  
  $$O(n)$$

- Balanced BST search:  
  $$O(\log n)$$
- Merge Sort recurrence:  
  $$T(n) = 2T\!\left(\frac{n}{2}\right) + O(n) = O(n \log n)$$
